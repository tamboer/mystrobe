Edit order is a good example for MyStroBe data manipulation transaction support.

First one can change order details and than the order lines info. Data is only sent to the Quarix server on the last tab as a single transaction. Even though separate data objects are used for order and order lines changed data is sent together.

'Edit Order' is an example of a custom edit record form. It does not use any of the framework built in components and can be used as a startup for any custom forms.

protected class EditOrderForm extends Form<Order> {
..

  FeedbackPanel feedback = new FeedbackPanel("feedback");
  feedback.setOutputMarkupId(true);
  add(feedback);

  TextField<String> instructionsTxt = new TextField<String>(OrderSchema.Cols.INSTRUCTIONS.id());
  add(instructionsTxt);

  TextField<String> carrierTxt = new TextField<String>(OrderSchema.Cols.CARRIER.id());
  add(carrierTxt);

..
