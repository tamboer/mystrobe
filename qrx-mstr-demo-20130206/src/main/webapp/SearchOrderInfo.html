Search Order uses MyStroBe data object link parameters to connect selected order with order lines display.

Connection is made through the component linker by providing the link parameters:

..
orderDo = new OrderDataObject();
orderLineDo = new OrderLineDataObject();
..
DataLinkParameters orderOrderLineLink = new DataLinkParameters(OrderSchema.Cols.ORDERNUM.id());
Set<DataLinkParameters> linkParams = new HashSet<DataLinkParameters>(1);
linkParams.add( orderOrderLineLink);
ComponentLinker.bindData(orderDo, orderLineDo, linkParams);
..

On click event on order tables uses ajax to reload order lines table. MyStroBe components expose methods that handle ajax events. One can easily extend default behavior or override it.

..
SimpleDataTableViewPanel<Order> tableView = new SimpleDataTableViewPanel<Order>
  ("orderViewId", orderFormConfig, 10, additionalColumns){
  @Override
  protected void onDataChanged(AjaxRequestTarget target) {
    super.onDataChanged(target);
    target.add(SearchOrderPage.this.get("orderLineNavigatorId"));
    target.add(SearchOrderPage.this.get("orderLineViewId"));
  }
};
