Customer page is build from MyStroBe standard components.

Customer details, table and navigator are built in components linked together through component linker. Sample code:

tableView = new SimpleDataTableViewPanel<Customer>("tableId", custFormConfig, 10, additionalColumns){.. };
ComponentLinker.bindDataTableData(customerDo, tableView);

pagesNavigator = new DataTablePagesNavigationPanel<Customer>("navigatorId", 10, 4) {..};
ComponentLinker.bindDataTableNavigation(pagesNavigator, customerDo);

custForm = new DynamicFormDataViewPanel<Customer>( "dynamicForm" , new Customer() , custDetailsConfig);
ComponentLinker.bindData( customerDo, custForm, null);
..

Wicket html for MyStroBe components:

..
<div wicket:id="dynamicForm"></div>
<div wicket:id="tableId"></div>
<div wicket:id="navigatorId"></div>
..

Modal window uses standard MyStroBe edit record panel, configured for lookup fields, required fields and others.

Configuration has multiple options and offers support for all details off the edit record panel. Same configuration class is used for both table and edit record form. One can specify the visible columns, order of columns, labels, column width, required fields, field types and many others.

For table component you can also configure additional columns using IMyStrobeColumn which inherits from wicket IColumn.
